<feature>
  <name>Server Game Logic: rotation, place, return, win, turns, getPublicState extension</name>
  <files>server/src/game.js, server/src/game.test.js</files>
  <behavior>
    rotateCells(cells, rotation):
    - [0,0],[0,1],[1,0] at 0° → [0,0],[0,1],[1,0] (unchanged)
    - [0,0],[0,1],[1,1] at 90° CW → normalize min to [0,0] → [0,1],[1,1],[1,0] (L rotated)
    - applying 4× 90° CW returns to original
    - negative offsets never appear in output (always normalized)

    placePiece(lobby, shapeId, rotation, originRow, originCol):
    - returns { ok: true, win: false } when all cells are empty and in-bounds
    - returns { ok: false, error: 'Cell occupied' } when any target cell is filled
    - returns { ok: false, error: 'Piece out of bounds' } when any target cell is outside gridSize
    - returns { ok: false, error: 'Shape already placed' } when shapeId is already on grid
    - returns { ok: false, error: 'Invalid shape' } when shapeId is not a movable shape in puzzle
    - after success: all rotated cells are written as { shapeId, movable: true } in lobby.grid
    - returns { ok: true, win: true } when placement completes the solution (checkWin passes)

    returnPiece(lobby, shapeId):
    - returns { ok: true } and clears all matching cells (movable, same shapeId) from lobby.grid
    - returns { ok: false, error: 'Shape not on grid' } when shapeId is not currently placed
    - returns { ok: false, error: 'Invalid shape' } when shapeId is not a movable shape

    checkWin(lobby, puzzle):
    - returns false when any non-null solution cell has null in grid
    - returns false when any non-null solution cell has a different shapeId in grid
    - returns false when any null solution cell has something in grid
    - returns true when every cell matches solution exactly
    - is NEVER called with solution exposed to client (internal server function only)

    advanceTurn(lobby):
    - increments activeTurnIndex by 1, wraps around modulo players.length
    - no-op when players.length === 0

    advanceTurnIfActive(lobby, socketId) [replaces stub]:
    - if disconnecting player IS the active player:
      - newLength = players.length - 1
      - if newLength === 0: activeTurnIndex = 0
      - else: activeTurnIndex = activeTurnIndex % newLength
    - if disconnecting player is NOT active and their index < activeTurnIndex:
      - activeTurnIndex = Math.max(0, activeTurnIndex - 1)
    - if disconnecting player is NOT active and their index >= activeTurnIndex:
      - no change

    getPublicState(roomCode) extension:
    - output includes activePlayerName (players[activeTurnIndex].name or null)
    - output includes activeTurnIndex (number, 0 if undefined)
    - output includes bankShapes: movable shapes from puzzle.shapes whose id does NOT appear
      as a { movable: true } cell in lobby.grid — each as { id, cells }
    - output NEVER includes solution or any field derived from puzzle.solution
  </behavior>
