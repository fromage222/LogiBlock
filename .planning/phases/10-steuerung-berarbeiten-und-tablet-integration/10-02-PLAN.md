**Document-level touchmove handler (wired ONCE, outside renderGrid/renderBank — add after the R key handler):**

```javascript
// ── Touch: drag-to-preview (document-level, wired once) ─────────────────────
document.addEventListener('touchmove', (e) => {
  if (!selectedShapeId) return;
  if (!gameScreen.classList.contains('active')) return;
  const touch = e.touches[0];
  const el = document.elementFromPoint(touch.clientX, touch.clientY);
  if (!el) return;

  // Hide cursor piece during touch drag (finger covers it)
  const cursorEl = getCursorEl();
  if (cursorEl) cursorEl.style.display = 'none';

  const row = parseInt(el.dataset.row);
  const col = parseInt(el.dataset.col);
  if (isNaN(row) || isNaN(col)) {
    // Finger moved off grid — clear ghost
    clearGhostPreview();
    touchDragging = false;
    return;
  }
  if (el.classList.contains('inactive')) return; // Pitfall 4: skip inactive cells
  touchDragging = true;
  lastHoveredRow = row;
  lastHoveredCol = col;
  updateGhostPreview(row, col);
}, { passive: true });
```

**Long-press return on grid cells (inside renderGrid(), on each placed movable cell):**

Inside `renderGrid()`, AFTER the existing `cell.addEventListener('click', ...)` block (the one from Plan 01), add touch handlers for placed movable cells:

```javascript
// Touch: long-press return for placed movable pieces
if (content && content.movable !== false && !content.inactive) {
  cell.addEventListener('touchstart', () => {
    if (selectedShapeId) return; // only long-press return when no piece selected
    clearTimeout(longPressTimer);
    longPressTimer = setTimeout(() => {
      handleReturnClick(content.shapeId);
      longPressTimer = null;
    }, 500);
  }, { passive: true });

  cell.addEventListener('touchend', () => {
    clearTimeout(longPressTimer);
    longPressTimer = null;
  });

  cell.addEventListener('touchmove', () => {
    // Finger moved — cancel long press (it became a drag)
    clearTimeout(longPressTimer);
    longPressTimer = null;
  });
}
```

**Important placement:** The `longPressTimer` variable is module-level (declared once at top), NOT per-cell. This prevents stale timer leaks when renderGrid() rebuilds DOM on state updates.

**Touch deselect — finger drags entirely off grid to non-game area:**

The existing document-level `click` deselect handler (from Plan 01, the one with `e.target.closest(...)`) already handles this case. When the user lifts their finger off a non-grid area, the browser fires a synthetic `click` event on that element. If it's not inside `#game-grid`, `#piece-bank`, or `#rotation-controls`, the deselect handler fires. No additional code is needed.

**Guard: prevent touch handlers from interfering on non-game screens (Pitfall 6):**

The `touchmove` handler already checks `gameScreen.classList.contains('active')`. The bank `touchstart` is only wired inside `renderBank()`, which only runs during game. The long-press handlers are per-cell inside `renderGrid()`, which is also game-only. No additional guards needed.
